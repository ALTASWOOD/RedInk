## 版本命名规范

### 语义化版本号

采用语义化版本号规范 (Semantic Versioning)：**MAJOR.MINOR.PATCH**

| 版本位 | 说明 | 示例 |
|--------|------|------|
| MAJOR | 重大版本，不兼容的API变更 | 1.0.0 → 2.0.0 |
| MINOR | 次要版本，向后兼容的新功能 | 1.0.0 → 1.1.0 |
| PATCH | 补丁版本，向后兼容的问题修复 | 1.0.0 → 1.0.1 |

### 预发布版本标识

| 标识 | 说明 | 示例 |
|------|------|------|
| alpha | 内部测试版，功能不完整 | 1.0.0-alpha.1 |
| beta | 公开测试版，功能基本完整 | 1.0.0-beta.1 |
| rc | 候选发布版，准备正式发布 | 1.0.0-rc.1 |

## 版本发布记录

### v1.0.0 (计划中)

**发布日期**：2026-10-10（计划）

**发布类型**：正式发布

**版本亮点**：
- 🎉 RedInk v1.0.0 正式发布
- 🛡️ 公私隔离设计，安全可控的AI公文助手
- 🤖 支持公网AI和本地AI双模式
- 📝 内置公文模板，符合国家标准
- 📚 知识库功能，智能问答

**主要功能**：

| 模块 | 功能 | 状态 |
|------|------|------|
| 公文编写 | 模板管理、AI辅助编写、多格式导出 | 计划中 |
| 文档管理 | 列表管理、分类、版本管理 | 计划中 |
| 素材库 | 短语、段落、图片、表格 | 计划中 |
| AI集成 | 公网AI、本地Ollama、环境切换 | 计划中 |
| 知识库 | 向量存储、RAG问答 | 计划中 |
| AI审核 | 内容审核、格式审核、报告 | 计划中 |
| 图表 | 流程图、架构图、AI生成 | 计划中 |

**系统要求**：
- Windows 10/11 (64位)
- macOS 12+ (Apple Silicon / Intel)
- 内存：8GB+ (本地AI需16GB+)
- 存储：500MB+ 安装空间
