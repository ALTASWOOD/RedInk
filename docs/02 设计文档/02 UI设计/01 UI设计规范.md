## 1. 设计原则

### 1.1 核心设计原则

| 原则 | 说明 |
|------|------|
| 环境感知优先 | 用户必须在任何时刻都能清晰识别当前环境（公网/私域） |
| 简洁专业 | 界面风格简洁大方，符合政务办公场景 |
| 高效操作 | 常用功能触手可及，减少操作步骤 |
| 一致性 | 保持交互方式和视觉风格的统一 |
| 无障碍 | 支持键盘操作，满足无障碍访问需求 |

### 1.2 设计关键词

```
专业  |  安全  |  高效  |  清晰  |  可信赖
```

## 3. 字体规范

### 3.1 字体选择

| 平台 | 中文字体 | 英文字体 | 等宽字体 |
|------|----------|----------|----------|
| Windows | 微软雅黑 | Segoe UI | Consolas |
| macOS | 苹方 | -apple-system | SF Mono |
| Linux | 文泉驿微米黑 | Ubuntu | Ubuntu Mono |

### 3.2 字体堆栈

```css
/* 正文字体 */
--font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto,
  'Helvetica Neue', Arial, 'Noto Sans', sans-serif, 'Apple Color Emoji',
  'Segoe UI Emoji', 'Segoe UI Symbol', 'Noto Color Emoji';

/* 中文字体 */
--font-family-cn: 'PingFang SC', 'Microsoft YaHei', '微软雅黑',
  'Hiragino Sans GB', sans-serif;

/* 等宽字体（代码） */
--font-family-code: 'SFMono-Regular', Consolas, 'Liberation Mono',
  Menlo, Courier, monospace;
```

## 4. 图标规范

### 4.1 图标风格

- **风格**: 线性图标为主，填充图标用于强调
- **尺寸**: 基础尺寸 16px，可选 12px、20px、24px
- **描边**: 统一 1.5px 描边粗细
- **圆角**: 统一 2px 圆角

### 4.2 环境标识图标

| 环境 | 图标 | 说明 |
|------|------|------|
| 公网 | 云朵 (Cloud) | 象征云端、互联网 |

### 5.3 活动栏规范

活动栏 (Activity Bar) 是主界面最左侧的垂直图标栏，用于快速切换功能面板。

#### 尺寸与布局

| 属性 | 数值 | 说明 |
|------|------|------|
| 宽度 | 48px | 固定宽度 |
| 图标尺寸 | 24px | 图标大小 |
| 图标间距 | 4px | 图标之间的垂直间距 |
| 图标容器 | 40px × 40px | 点击区域 |
| 上下内边距 | 8px | 顶部和底部留白 |

#### 图标配置

| 位置 | 图标 | 功能 | 说明 |
|------|------|------|------|
| 上部 | 📄 | 我的文档 | 文档管理列表 |
| 上部 | 📋 | 模板库 | 公文模板浏览 |
| 上部 | 🕐 | 最近使用 | 最近编辑的文档 |
| 上部 | ⭐ | 收藏 | 收藏的文档和模板 |
| 上部 | 📦 | 素材库 | 常用短语、段落模板、图片素材 |
| 上部 | 📊 | 图表 | 图表绘制工具 |
| 底部 | ⚙️ | 设置 | 应用设置（固定底部） |

#### 交互行为

| 行为 | 描述 |
|------|------|
| 单击 | 切换到对应功能面板，侧边栏显示相应内容 |
| 悬停 | 显示背景色和 Tooltip 提示功能名称 |
| 选中 | 显示左侧指示条（2px宽，主题色），背景高亮 |
| 再次点击选中项 | 折叠/展开侧边栏 |

#### 样式规范

```css
.activity-bar-item:hover {
  background: #F5F5F5;

/* 选中状态 */
.activity-bar-item.active {

  width: 2px;
  background: var(--theme-color);
  border-radius: 0 1px 1px 0;
}
```

## 7. 动效规范

### 7.1 动效原则

| 原则 | 说明 |
|------|------|
| 自然流畅 | 动画曲线符合物理规律 |
| 适度克制 | 避免过度动画，影响效率 |
| 有意义 | 动画应传达信息或引导注意 |
| 一致性 | 相同类型操作使用相同动画 |

### 7.2 动效时长

| 类型 | 时长 | 用途 |
|------|------|------|
| 即时反馈 | 100ms | 按钮点击、开关切换 |
| 标准过渡 | 200ms | 元素出现/消失 |
| 复杂动画 | 300ms | 面板展开/收起 |
| 页面转场 | 400ms | 页面切换 |

### 7.3 缓动函数

```css
/* 标准缓动 */
--ease-standard: cubic-bezier(0.4, 0, 0.2, 1);

/* 减速缓动（进入） */
--ease-decelerate: cubic-bezier(0, 0, 0.2, 1);

/* 加速缓动（离开） */
--ease-accelerate: cubic-bezier(0.4, 0, 1, 1);
```

### 7.4 环境切换动效

```
环境切换动画序列:

1. 状态栏颜色渐变 (300ms)
   旧颜色 ──────────────> 新颜色
   
2. 边框颜色同步变化 (300ms)

3. 图标切换动画 (200ms)
   旧图标淡出 → 新图标淡入
```

## 9. 无障碍设计

### 9.1 对比度要求

| 类型 | 最低对比度 | 说明 |
|------|------------|------|
| 正常文字 | 4.5:1 | WCAG AA标准 |
| 大号文字 | 3:1 | 18px以上或14px加粗 |
| 图标/图形 | 3:1 | 重要功能图标 |

### 9.2 屏幕阅读器支持

- 所有图标提供 aria-label
- 表单元素关联 label
- 动态内容使用 aria-live
- 提供跳过导航链接

### 9.3 色彩无障碍

- 不单独依赖颜色传达信息
- 重要信息同时使用图标/文字
- 支持高对比度模式
