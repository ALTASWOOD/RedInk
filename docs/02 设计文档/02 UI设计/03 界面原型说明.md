### 1.2 活动栏说明

活动栏 (Activity Bar) 位于界面最左侧，提供主要功能入口的快速切换。

| 图标 | 功能 | 说明 |
|------|------|------|
| 📄 | 我的文档 | 查看和管理所有文档 |
| 📋 | 模板库 | 浏览和使用公文模板 |
| 🕐 | 最近使用 | 快速访问最近编辑的文档 |
| ⭐ | 收藏 | 查看收藏的文档和模板 |
| 📦 | 素材库 | 管理常用短语、段落模板、图片素材 |
| 📊 | 图表 | 图表绘制工具 |
| ⚙️ | 设置 | 应用设置（位于底部） |

**交互行为：**
- 点击图标切换侧边栏显示的内容
- 当前选中项显示高亮背景和左侧指示条
- 悬停显示功能名称 Tooltip
- 设置图标固定在底部
