## 1. AI 功能概述

RedInk 集成了强大的 AI 能力，帮助您更高效地撰写公文。

| 功能 | 说明 |
|------|------|
| 内容生成 | 根据要点自动生成公文 |
| 内容审核 | 检查格式和内容规范性 |
| 内容改写 | 润色、精简或扩展文本 |
| 智能对话 | 自由咨询写作问题 |
| 图表生成 | 用自然语言描述生成图表 |

## 2. 内容生成

### 2.1 使用步骤

1. **选择类型**：在生成面板选择公文类型（通知、报告、请示等）

2. **输入要点**：
   ```
   • 召开年度工作总结会议
   • 时间：3月15日上午9点
   • 地点：A会议室
   • 要求各部门负责人参加
   ```

3. **补充说明**（可选）：
   ```
   语气正式，强调会议重要性
   ```

4. **点击生成**：等待 AI 生成内容

5. **查看结果**：内容以流式方式逐字显示

6. **插入或重生成**：满意则插入，否则重新生成

### 2.2 技巧

- 要点越详细，生成内容越准确
- 可多次生成，选择最佳版本
- 生成后仍可手动修改完善

## 3. 内容审核

### 3.1 审核类型

| 类型 | 检查内容 |
|------|----------|
| 格式审核 | 标题字号、正文格式、落款、页边距等 |
| 内容审核 | 逻辑结构、语言表达、用词规范 |
| 全面审核 | 以上全部 |

### 3.2 问题级别

| 级别 | 图标 | 说明 |
|------|------|------|
| 高 | 🔴 | 必须修改 |
| 中 | 🟡 | 建议修改 |
| 低 | 🟢 | 可选修改 |

### 3.3 使用技巧

- 完成初稿后进行审核
- 优先处理高优先级问题
- 点击"定位"快速跳转
- 使用"一键修复"自动修正

## 4. 内容改写

### 4.1 改写风格

| 风格 | 效果 |
|------|------|
| 润色 | 提升文采，更流畅优美 |
| 精简 | 删减冗余，更简洁有力 |
| 扩展 | 丰富内容，更详细充实 |
| 正式化 | 转为书面语，更正式规范 |

### 4.2 使用方法

1. 选中需要改写的文本
2. 选择改写风格
3. 查看对比结果
4. 确认替换或取消

## 6. AI 图表生成

### 6.1 使用方法

1. 选择图表类型
2. 用自然语言描述：
   ```
   组织架构：公司下设研发部、市场部、财务部，
   研发部有前端组和后端组
   ```
3. AI 自动生成图表
