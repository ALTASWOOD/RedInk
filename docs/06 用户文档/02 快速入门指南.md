# 快速入门指南

## 欢迎使用 RedInk

RedInk 是一款 AI 驱动的公文报告编写软件，帮助您高效撰写规范的公文。

---

## 1. 首次启动

### 1.1 环境选择

首次启动时，您会看到环境选择界面：

| 环境 | 图标 | 说明 | 适用场景 |
|------|------|------|----------|
| 🛡️ 私域环境 | 绿色 | 数据本地处理，完全离线 | 涉密文档、内部文件 |
| ☁️ 公网环境 | 蓝色 | 使用云端 AI 服务 | 一般性文档 |

**建议**：处理敏感内容时请选择私域环境。

### 1.2 界面概览

```

│  🛡️ 私域环境                           [切换到公网] [设置]      │

│          │                                      │               │

│  列表    │                                      │               │
```

## 2. 创建第一篇公文

### 步骤 1：新建文档

1. 点击左上角 **[+ 新建]** 按钮
2. 选择公文类型（如：通知）
3. 可选择使用模板

### 步骤 2：使用 AI 生成内容

1. 在右侧 AI 面板点击 **[生成]**
2. 选择文档类型
3. 输入关键要点：
   - 会议时间
   - 会议地点
   - 参会人员
4. 点击 **[生成公文]**
5. 等待 AI 生成内容
6. 点击 **[插入到文档]**

### 步骤 3：编辑完善

- 直接在编辑区修改内容
- 使用工具栏调整格式
- 可随时按 `Ctrl+S` 保存

### 步骤 4：AI 审核

1. 点击 **[审核]** 标签
2. 选择审核类型（格式/内容/全部）
3. 点击 **[开始审核]**
4. 查看问题列表
5. 点击 **[定位]** 跳转到问题位置
6. 使用 **[一键修复]** 自动修正

### 步骤 5：导出

1. 点击工具栏 **[导出]**
2. 选择格式（Word/PDF）
3. 选择保存位置
4. 点击 **[导出]**

## 3. 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+N` | 新建文档 |
| `Ctrl+S` | 保存 |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Y` | 重做 |
| `Ctrl+G` | AI 生成 |
| `Ctrl+R` | AI 审核 |
| `Ctrl+E` | 切换环境 |
