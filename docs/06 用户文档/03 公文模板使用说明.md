## 1. 模板概述

RedInk 提供多种预设公文模板，帮助您快速创建规范的公文。

### 1.1 内置模板分类

| 分类 | 模板 | 用途 |
|------|------|------|
| 通知 | 通用通知、会议通知、工作通知 | 传达事项、布置工作 |
| 报告 | 工作报告、调研报告、专项报告 | 汇报工作、反映情况 |
| 请示 | 一般请示、经费请示 | 请求上级批准 |
| 批复 | 批复模板 | 答复下级请示 |
| 函 | 商洽函、复函、催办函 | 平级单位往来 |
| 会议纪要 | 会议纪要模板 | 记录会议内容 |

---

## 2. 使用内置模板

### 2.1 从模板库选择

1. 点击左侧 **[模板库]**
2. 浏览分类或搜索
3. 预览模板内容
4. 点击 **[使用]**
5. 系统创建新文档并应用模板

### 2.2 新建时选择模板

1. 点击 **[+ 新建]**
2. 选择 **[使用模板]**
3. 选择模板
4. 输入文档标题
5. 点击 **[创建]**

## 3. 模板结构说明

### 3.1 通知模板

```
                关于XXX的通知

各部门、各单位：

    [正文内容]

    一、[第一部分]

    二、[第二部分]

    特此通知。


                              [发文单位]
                           XXXX年XX月XX日
```

### 3.2 报告模板

```
                关于XXX的报告

[主送机关]：

    [导语]

    一、基本情况
    
    二、主要做法
    
    三、存在问题
    
    四、下步计划

                              [发文单位]
                           XXXX年XX月XX日
```

### 3.3 请示模板

```
                关于XXX的请示

[主送机关]：

    [请示事项和理由]

    妥否，请批示。


                              [发文单位]
                           XXXX年XX月XX日
```

## 4. 自定义模板

### 4.1 创建自定义模板

1. 编辑好文档格式和结构
2. 点击菜单 **[文件]** > **[另存为模板]**
3. 填写模板信息：
   - 模板名称
   - 分类
   - 描述
4. 点击 **[保存]**

### 4.2 管理自定义模板

- **编辑**：模板库 > 我的模板 > 点击编辑图标
- **删除**：模板库 > 我的模板 > 点击删除图标
- **收藏**：点击模板的收藏图标，方便快速访问

## 5. 模板格式规范

### 5.1 字体字号

| 元素 | 字体 | 字号 |
|------|------|------|
| 标题 | 方正小标宋简体 | 二号 (22pt) |
| 主送机关 | 仿宋_GB2312 | 三号 (16pt) |
| 正文 | 仿宋_GB2312 | 三号 (16pt) |
| 一级标题 | 黑体 | 三号 (16pt) |
| 二级标题 | 楷体_GB2312 | 三号 (16pt) |
| 落款 | 仿宋_GB2312 | 三号 (16pt) |

### 5.2 页面设置

| 项目 | 规格 |
|------|------|
| 纸张 | A4 (210mm × 297mm) |
| 上边距 | 37mm |
| 下边距 | 35mm |
| 左边距 | 28mm |
| 右边距 | 26mm |
| 行距 | 28pt |

## 6. 使用技巧

### 6.1 快速填充

- 占位符会高亮显示
- 点击占位符直接输入
- Tab 键跳转下一个占位符

### 6.2 结合 AI 使用

1. 选择模板创建文档
2. 在 AI 面板输入要点
3. AI 生成内容自动填充模板

### 6.3 批量使用

对于需要发给多个单位的通知：
1. 完成内容编辑
2. 修改主送机关
3. 导出
4. 重复 2-3 步
