## 1. 公文种类

### 1.1 法定公文种类

| 种类 | 适用范围 |
|------|----------|
| 决议 | 经会议讨论通过的重要决策事项 |
| 决定 | 对重要事项或重大行动作出安排 |
| 命令 | 依据法律发布行政法规和规章 |
| 公报 | 公布重要决定或重大事件 |
| 公告 | 向社会公布重要事项或法定事项 |
| 通告 | 公布社会各方面应遵守的事项 |
| 意见 | 对重要问题提出见解和处理办法 |
| 通知 | 发布、传达要求下级机关执行 |
| 通报 | 表彰先进、批评错误、传达情况 |
| 报告 | 向上级汇报工作、反映情况 |
| 请示 | 向上级请求指示、批准 |
| 批复 | 答复下级机关请示事项 |
| 议案 | 提请审议事项 |
| 函 | 不相隶属机关之间商洽工作 |
| 纪要 | 记载会议主要情况和议定事项 |

| 项目 | 规格 |
|------|------|
| 纸张规格 | A4 (210mm × 297mm) |
| 上白边 | 37mm ± 1mm |
| 下白边 | 35mm ± 1mm |
| 左白边 | 28mm ± 1mm |
| 右白边 | 26mm ± 1mm |

## 4. 标题层次

### 4.1 结构层次序数

```
第一层    一、
第二层    (一)
第三层    1.
第四层    (1)
```

### 4.2 标题格式

| 层次 | 格式 | 字体 |
|------|------|------|
| 一级标题 | 一、XXX | 黑体 |
| 二级标题 | (一)XXX | 楷体 |
| 三级标题 | 1.XXX | 仿宋 |
| 四级标题 | (1)XXX | 仿宋 |

---

## 5. 成文日期

### 5.1 格式要求

- 使用汉字数字书写：二〇二六年三月十五日
- 或使用阿拉伯数字：2026年3月15日

### 5.2 位置要求

- 在发文机关署名下一行
- 右空四字编排
- 与发文机关署名居中对齐

## 6. 常见格式问题

| 问题 | 规范要求 |
|------|----------|
| 标题字号过小 | 应为二号字 |
| 正文首行未缩进 | 首行缩进两字 |
| 落款日期格式不统一 | 统一使用规定格式 |
| 行距不规范 | 应为28磅固定行距 |
| 页边距不正确 | 按规定设置页边距 |
