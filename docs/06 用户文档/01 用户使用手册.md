## 1. 产品介绍

RedInk 是一款 AI 驱动的公文报告编写软件，提供智能写作、格式审核、图表绘制等功能，帮助您高效撰写规范的公文。

### 核心特性

- 🤖 **AI 智能写作** - 根据要点快速生成公文
- ✅ **格式审核** - 自动检查公文格式规范性
- 🛡️ **公私隔离** - 敏感文档本地处理，数据安全
- 📊 **图表绘制** - 流程图、组织架构图等
- 📤 **多格式导出** - Word、PDF、OFD 等

## 3. 文档管理

### 3.1 新建文档

1. 点击 **[+ 新建]** 按钮
2. 选择新建方式：
   - **空白文档** - 从零开始
   - **使用模板** - 选择预设模板
3. 选择文档类型
4. 输入标题
5. 点击 **[创建]**

### 3.2 打开文档

- 双击文档列表中的文档
- 或右键选择 **[打开]**

### 3.3 保存文档

- 手动保存：点击 **[保存]** 或按 `Ctrl+S`
- 自动保存：默认每30秒自动保存

### 3.4 删除文档

1. 右键点击文档
2. 选择 **[移至回收站]**
3. 可从回收站恢复或永久删除

## 4. 编辑功能

### 4.1 文本编辑

- 支持标准文本操作（复制、粘贴、撤销等）
- 支持格式设置（加粗、斜体、字号等）

### 4.2 格式设置

| 功能 | 操作 |
|------|------|
| 加粗 | `Ctrl+B` 或工具栏按钮 |
| 斜体 | `Ctrl+I` |
| 下划线 | `Ctrl+U` |
| 字号调整 | 工具栏下拉选择 |

### 4.3 公文格式

点击 **[格式]** 菜单可应用：
- 一级标题（黑体 16pt）
- 二级标题（黑体 14pt）
- 正文（仿宋 14pt）
- 落款格式

## 5. AI 功能

### 5.1 内容生成

1. 点击AI面板 **[生成]** 标签
2. 选择公文类型
3. 输入关键要点（每行一个）
4. 补充说明（可选）
5. 点击 **[生成]**
6. 查看生成结果
7. 点击 **[插入到文档]** 或 **[重新生成]**

### 5.2 内容审核

1. 点击 **[审核]** 标签
2. 选择审核类型：
   - 格式审核 - 检查排版规范
   - 内容审核 - 检查逻辑表达
   - 全面审核 - 两者都检查
3. 点击 **[开始审核]**
4. 查看问题列表
5. 点击问题项的 **[定位]** 跳转到对应位置
6. 使用 **[一键修复]** 自动修正可修复项

### 5.3 内容改写

1. 选中需要改写的文本
2. 点击 **[改写]** 标签
3. 选择改写风格：
   - 润色 - 提升文采
   - 精简 - 缩减篇幅
   - 扩展 - 丰富内容
   - 正式化 - 更正式的表达
4. 查看改写结果
5. 选择 **[替换]** 或 **[取消]**

## 6. 模板使用

### 6.1 使用内置模板

1. 点击左侧 **[模板库]**
2. 浏览或搜索模板
3. 点击 **[使用]** 创建新文档

### 6.2 创建自定义模板

1. 编辑好文档格式
2. 点击 **[另存为模板]**
3. 填写模板名称和描述
4. 选择分类
5. 保存

---

## 7. 图表绘制

### 7.1 插入图表

1. 将光标定位到插入位置
2. 点击 **[插入图表]**
3. 选择图表类型
4. 编辑图表内容
5. 点击 **[插入]**

### 7.2 支持的图表类型

| 类型 | 用途 |
|------|------|
| 流程图 | 工作流程、审批流程 |
| 组织架构图 | 部门结构、人员关系 |
| 思维导图 | 内容规划、要点梳理 |
| 时间轴 | 项目进度、历史沿革 |

### 7.3 AI 生成图表

1. 选择图表类型
2. 输入描述（如："公司有3个部门..."）
3. 点击 **[AI生成]**

## 8. 导出功能

### 8.1 导出格式

| 格式 | 说明 |
|------|------|
| DOCX | Microsoft Word 格式 |
| PDF | 便于分发和打印 |
| MD | Markdown 格式 |
| OFD | 版式文档格式 |

### 8.2 导出选项

- **使用标准公文格式** - 符合 GB/T 9704 标准
- **嵌入字体** - 确保在其他电脑正确显示
- **包含修订历史** - 保留修改记录

## 10. 数据安全

### 10.1 公私隔离

- **私域环境**：所有数据本地处理，不联网
- **公网环境**：输入内容会发送至云端 AI

### 10.2 敏感检测

系统会自动检测：
- 身份证号
- 手机号码
- 银行卡号
- 涉密关键词

检测到敏感信息时会警告提示。

### 10.3 数据备份

- 数据存储在本地
- 建议定期备份数据目录
